## How to use
#### Server Side
1 - Install server side dependencies
  ```
  npm install
  ```
2 - Run server
  ```
  npm start
  ```
#### Client Side
1 - Install client side dependencies
  ```
  cd client
  npm install
  ```
2 - Run client
  ```
  npm start
  ```

## Environment variables
Add your config variables values in the .env file in backend/config folder.
```
  PORT=5000
  DATABASE_LINK=""
```
